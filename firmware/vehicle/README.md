# Firmware vehicle

Proyecto ESP-IDF para el ESP32-S3 montado en el PiCar-X.

Responsabilidades:

- MCPWM para motores.
- Sensor de distancia.
- Logica adaptativa de velocidad.
- Comunicacion WiFi con el mando.
- Telemetria para depuracion o panel web.

Estado actual: scaffold con tareas FreeRTOS y stubs de hardware.
